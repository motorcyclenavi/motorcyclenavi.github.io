이륜차 오프라인 GPS 내비!
https://play.google.com/store/apps/details?id=com.offline.navi.motorcamp&hl=ko

오프라인에서 GPS로 이륜차 길을 찾다.

-이 앱은 위성 GPS만을 이용하여 위치를 찾는 실외 위주의 내비 앱입니다.
 
*위치 찿기 - 위성 및 GPS 미수신시 네트워크를 이용

-음성 안내 서비스
전화기 글자읽어주기 설정에서 구글 또는 삼성 음성 서비스가 한국어 또는 영어로 설치되어 있어야합니다.그렇치 않다면 구글 목소리 데이터는 설정에서, 삼성은 갤럭시 스토어에서 다운받아야합니다	

1. 차량 종류 (V2)
-빠른 라이더들을 위한 모토싸이클 - 자동차 전용/이륜차 진입 금지 제외.
-느린 라이더들을 위한 스쿠터(베타) -  + 80 km 구간 제외.
-더느린 라이더들을 위한 스쿠터 50cc (개발중) -  + 70 km 구간 제외.
-자전거 라이더들을 위한 자전거(베타) -  + 대한민국 모든 자전거 전용 도로 

2. 길 프로파일(V2)
-짧고 빠른 길: 운행 거리와 시간을 적절히 계산해서 안내
-짧은 길: 운행 거리를 줄일 수 있는 길로 안내
-빠른 길: 운행 시간을 줄일 수 있는 길로 안내
-커브 길: 라이딩이 재미 있는 커브 길로 안내

3. 지도
사용자 지도 설정 
- 배경 색상
- 텍스처
- 도로 - 글자 색상, 자전거길 색상, 고속도로 색상 ,이륜차 진입금지 색상 + 너비 
- 관심 지점 - 지도상에 표시 / 미표시 설정: 경계선, 지역 이름, 주유소, 충전소, 야영장, 화장실...
 
4. 길찾기  및 길안내
-내비게이션모드시 지도고정/내비모드 버튼
-자기장 방위 -설정->위치->자기장 방위; 위성 미연결시에도 자기장으로 방향 찾기
-미리보기 가상운행(V1)- 속도, 운행 모드, 주변 검색
-내비게이션 모드(V2) - 고정 운행 화면, 방위 화살표, 가까운 줌/높은 속도에서의 개선된 위치 표시 움직임 및 프레임.
-턴바이턴 안내 (v2)
	시각 -다음 회전과 그 다음 지점에 색깔 회전 / 거리 표시
	소리 -일정한 거리 간격을 두고 다음 회전 알림 
	(구간초,1000m,500m,300m,200m,100m)	
-자동차 전용 /이륜차진입금지 도로를 제외하고 뱃길을 포함한 모든 도로를 검색.
-회전 금지 구간 회피.
-자동재탐색 (V4) 운행중 길을 벗어나면 자동 재탐색. 더 정확해진 재탐색 거리; 위성/자기장 방위 방향으로 수동/자동 길 탐색(빠른길/짧고빠른길 프로파일 적용)
-자동차 전용도로 / 이륜차 진입 금지 지도상에 레이어 표시로 시각화.
-가상위치 기능으로 미리 길 찾기 및 보기(길 리스트에서 선택 가능)
-위치 수신 시간과 거리를 직접 조정 할 수 있습니다.
-내비 선 두께/색깔 : 자동 선 조절
-현위치/목적지  공유 버튼
-주소/좌표 공유 설정 추가 
-음성 안내시 음악소리 음소거
-안내 소리 크기 조절
-운행 정보 : GPS 속도, 도로 이름, 구간 거리, 실시간 속도로 목적지까지 계산되는 남은 시간 

5. 검색 (V2)
-온라인-장소, 주소, 좌표 , 오프라인-주소, 좌표
-주유소 검색 - 반경 5 km 내 가장 싼 주유소 및 리스트 ; 오프라인 저장 및 검색
-목적지검색 - 복수의 결과 검색 및 오프라인 저장
-오프라인 데이터베이스: 개방 화장실, 무료 야영장, 혼다/대림 오토바이 서비스 센터, 병원, 버스 터미널, 기차역, 여객선 터미널등
-지도에서 목적지 선택 확인버튼 누르면 또는 길게 누르면 주소 자동 저장

6. 자료 저장  
-장소 : 현 위치나 목적지를 간단하게 이름, 주소, 좌표, 그리고 사진과 같이 저장, CSV로 내보냄
-트랙 : 운행 하면서 트랙 좌표를 저장하고 CSV, GPX로 내보냄
-앱업데이트시 기존 저장된 자료 자동 백업 복구
-주유소: 검색 리스트 자동 저장
-주소: 검색 및 리스트에서 클릭한 주소 자동저장

7. 오프라인 
-인터냇이 느리거나 끊겨도 길을 찾을 수 있습니다

Motorcycle Navigation
https://play.google.com/store/apps/details?id=com.offline.navi.motorcamp

- This is GPS based navigation app for outdoor use

*This app requires GPS connection and/or network(Internet,WI-FI,Bluetooth) to find location

- Road data offlimit to motorcycle / route data updates on Nov. 25 2022

-Text to speech requirement for voice instruction
Google or Samsung text-to-speech engine should already be set up on the phone with Korean and English voice data installed. If not, please download directly from the setting for Google and from galaxy store for Samsung

1. Vehicles
-Motorcycle - excludes car only roads
-Scooter(beta) - plus excludes 80 km roads
-Scooter 50 cc (in development) - plus excludes 70 km roads
-Bicycle(beta) - plus bicycle only roads

2. Routes
-Short and fastest : middle ground between shortest and fastest route. 
-Shortest : shortest calculated distance.
-Fastest : fastest calculated time.
-Curvature : more fun riding with curved roads.

3. Map - User Customization
- Background color
- Textures
- Roads layer - color: bicycle, freeway, and car only roads; width: car only roads 
- POI - on / off switch for boundary, area name, gas stations, camp grounds, restroom etc 

4. Route Search & Navigation
-fixed / navigation mode button
-magnetic direction: settings->location->magnetic; find direction with or without satellite connection
-Route drive preview - preview route in animated action with speed control and drive mode. Search using place button.
-navigation mode(V2) -  fixed screen and smoother location marker movement at high zoom / high speeds
-Turn by turn(V2) - visual, voice 
	Visual - Displays turn sign on the next turn point
	Voice  - Voice Notificatioin after certain distance until next turn
	(Start,1000m,500m,300m,200m,100m)	
-Excludes roads offlimit to motorcycle.
-Avoids turn restrictions.
-Auto-navigation function(V4) - improved off-route distance calculation for auto-rerouting; 
	manual/automatic route search towards GPS/magnetic direction (for fastest/short-fastest profiles only)
-View roads off limit to motorcycles.
-Search and preview route with user selected location    
-Control time/distance location update.
-Control thickness/visibility of navigation line.
-Share location(address/coordinates)
-Mute media when voice instruction is on
-set volume for voice instruction
-Driving Information: GPS speed, street name, address/coordinates, time remaining
-Gas Station - Search for the cheapest gas; automatically saved to offline list;

5. Search 
- Destination with name, address, or coordinates if online, coordinates,address if offline.
- Offline address search list
- Offline database includes public restrooms, free camp sites, honda/darim motorcycle repair shops, hospitals, bus terminals, train stations, and ship terminals.
- Gas Station - Search for the cheapest gas within 5 km radius; automatically saved to offline list.
- Save address by selecting destination.

6. Save and export  
-Place - Save places and export as CSV.
-Track - Record, save tracks while moving and export as GPX. 
-Gas station - auto saved
-Address - Automatically save searched address and clicked address from the list.

7. Offline - Find routes with slow or disconnected Internet.
